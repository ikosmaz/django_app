Notes for the web site

This webapp is a project made for Coursera Django for Everybody, and added some other features after the course.

Tags: Python, Django, Bootstrap, css, js, fontawsome, avatar

Logged in users can:

  • add an ad
  • comment on ads
  • send encrypted in app messages to the ad owner
  • use emojis on ad title, comments and messages
  • like ads and comments
  • rate others ads
  • edit/delete own ads and delete own comments
  • send an email to get reset password link
  • admin can manage users and ads on admin panel

Github account can be used to login (social login)

Up to 3MB photos can be added to ads

Search function works through all ad titles and ad details
